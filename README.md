# ТСН «ДОНИ» — информационный портал садоводства

**MVP · Июнь 2026**

---

## Что это

Официальный информационный портал ТСН «ДОНИ» (Пушкинский район, Санкт-Петербург). Единое окно для собственников участков: новости правления, финансы и взносы, документы и протоколы собраний, статус инфраструктуры.

## Для кого

Собственники участков, 35–65 лет. Доступ с компьютера и телефона.

## Что уже есть (MVP)

- **Главная страница** — важные объявления, быстрый доступ, последние новости
- **Новости** — публикации правления с архивом
- **Финансы** — взносы, реквизиты, список должников
- **Документы** — устав, протоколы, материалы собраний
- **Инфраструктура** — статус газификации, водоснабжения, благоустройства
- **Контакты** — адрес, телефоны правления
- **Поиск** по сайту
- **Админ-панель** — управление контентом без программиста

## Что нужно для продакшена

1. **Отдельный сервер** — 1 CPU, 1 GB RAM, 10 GB SSD (от 400 ₽/мес)
2. **Домен** — например, `tsn-doni.ru`
3. **SSL-сертификат** — бесплатный Let's Encrypt
4. **Наполнение контентом** — реальные новости, документы, реквизиты

## Дизайн

Природная палитра (forest green + wheat gold + sand), editorial-типографика (Fraunces + Newsreader). Без стандартных AI-шаблонов. Дизайн-аудит: 10 категорий, полный отчёт в `docs/design-audit-doni-2026-06-13.md` (доступен по запросу).
